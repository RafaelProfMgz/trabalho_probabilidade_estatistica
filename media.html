Média Aritmética

A média aritmética é a soma de todos os valores dividida pelo número total de observações.

x̄ = Σx / n

No mercado de hardware, ela indica o preço médio praticado no período, servindo como base para identificar se o preço atual está inflacionado ou em promoção.
