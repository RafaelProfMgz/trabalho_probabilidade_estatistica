Distribuição de Frequência

A Média Aritmética

Representa o valor central obtido pela soma de todos os elementos dividida pela quantidade de observações.

x̄ = Σx / n

No nosso caso, indica o preço médio praticado durante todo o período registrado.

A Mediana

É o valor que ocupa a posição central de um conjunto de dados ordenados.

Me = Posição (n + 1) / 2

Diferente da média, a mediana não é distorcida por picos de preço extremos.

A Moda

O valor que ocorre com a maior frequência absoluta no conjunto de dados.

Mo = Valor com maior contagem

Desvio Padrão

Indica a variabilidade ou dispersão dos dados em relação à média.

σ = √[ Σ(x - x̄)² / n ]

Um desvio alto indica um mercado volátil; um desvio baixo indica preços estáveis.

O Cálculo do Preço Ideal

Utilizamos uma inferência estatística para definir quando o preço está em uma oportunidade real de compra.

Preço Ideal = Mediana - (0.5 * Desvio Padrão)

Metodologia:

  • Mediana: Usada como âncora para evitar distorções de estoque.
  • Desvio Padrão: Consideramos a volatilidade para aceitar um desconto que seja estatisticamente possível.
  • Segurança: O sistema nunca recomenda um valor abaixo do mínimo histórico registrado.
